# Kumatan Dev Tools

Kumatan Converterの姉妹アプリとして、開発中によく使う変換・整形ツールをまとめたPWAです。

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

- Build Command: `npm run build`
- Output Directory: `dist`

## MVP Tools

- px ↔ rem
- 秒 ↔ ms
- HEX ↔ RGB
- Unix timestamp ↔ 日時
- JSON整形

## Icon

アプリアイコンは `public/icons/` に置いています。
